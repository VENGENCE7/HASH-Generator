# HASH-Generator #
This app allows  you to generate HASH of any string. <br />
It allows salting and iterations of salting. <br />
it also allows you to limit the length of the Hash. <br />
There are ***10 HASHES*** available:
  * MD5
  * BLAKE2B
  * BLAKE2S
  * SHA3_256
  * SHA3_512
  * SHA1
  * SHA224
  * SHA256
  * SHA384
  * SHA512
